# digital_finance_chatbot

## Dependency

scikit-learn, pysparnn, slackclient

## Preprocessing

Modify your bash
```
export SLACK_BOT_TOKEN='your bot user access token here'
``` 

## Usage

```
python3 chatbot.py
```
